Sobre este Portfólio

Este portfólio apresenta uma estrutura hierárquica de conhecimento, onde cada certificado blockchain representa um fragmento de criação — técnico, artístico ou espiritual. Os itens mais complexos (como Teses de Mestrado e o Framework MRCF) aparecem no topo de cada árvore temática.

Cada NFT está registrada em IPFS com hash SHA-256, verificável na blockchain Polygon e compatível com a rede social Sarafu Network para trocas com moeda digital.
